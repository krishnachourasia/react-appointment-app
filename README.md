## Pet Hospital Appointment System

In the project, you can do:

Search an apoointment.

Sort the list of apoointments based on different tags.

Add an appointment.

Delete an appointment.


## Libraries Used

React.js

Bootstrap 4

Lodash

Moment

React-moment

Popper.js
