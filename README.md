## Pet Hospital Appointment System

In the project, you can:

- Search an appointment.

- Sort the list of appointments based on different tags.

- Add an appointment.

- Delete an appointment.


## Libraries Used

- React.js

- Bootstrap 4

- Lodash

- Moment

- React-moment

- Popper.js

Clone this repo, and run
```
npm install 
```

And then, 
```
npm start
```
